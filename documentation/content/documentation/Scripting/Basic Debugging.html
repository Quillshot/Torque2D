Basic Debugging

When trying to figure out why your scripts aren't doing exactly what you intended them to do, there is one simple thing to try first: echo().

Echo can be used to print a string to the console while the game is running. This can be used to check if methods are being called or if the values of a variable are what you expect.

To access the console, press ~ in the top left of your keyboard inside your game or the editor. There will be lots of stuff there - the console reports errors with the loading or running of T2D or your game. Press ~ again to close the console. The console's text will also be saved in your console.log file in your install directory. This is invaluable if your game crashes because the log will usually have saved some information about what caused the crash.

To echo out a value, simply create a string with the values you want inside. For example:

echo("The %this object is " @ %this);

You can do more elaborate debugging in an advanced script editor like Torsion or CodeWeaver.
