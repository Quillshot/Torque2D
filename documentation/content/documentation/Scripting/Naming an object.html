Naming Objects

To change or set an object's name from the Level Builder, select the object.

Now, go to the Edit Tab (as shown in Figure 1.1.1).

Figure 1.1.1

Next, open the Scripting Section and type a name into the Name field (I called this object 'pTank') (as shown in Figure 1.1.2).

Figure 1.1.2





Brief Explanation of the Name field:

Once an object is named, it can be accesed anywhere in script by it's name. 
function up()
{
   pTank.setLinearVelocity(0,-10);
}

The name of the object simply resolves into the ObjectID given by the engine, but it's still a very easy way to reference common objects in your game, like a player.
