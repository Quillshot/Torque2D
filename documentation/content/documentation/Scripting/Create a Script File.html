Creating a Script File

Script editing

Those of you that are new to script editing will definitely want to read through this section. T2D scripts end with a ".cs" extension, such as "game.cs". Now, you may or may not see this extension when browsing for the files, depending on whether you're working on Windows or a Mac.


Windows scripting (Mac users skip to next section)


First you will need to browse out to the directory you saved your project to. Once you are there ensure you browse into your game project folder. From this folder you can a "common" and a "game" folder. Now you should either see a file called "main" as shown in the left picture(your icons images may be different than the picture displayed) or a file called "main.cs" as shown in the right picture.






If you see the "main.cs" file then you don't need to change any further settings, though if you only see "main" then you need to change an explorer setting that will allow you to view extensions. You can do this by clicking the Tools->Folder Options... menu.




You should then be presented with a Folder Options dialog. Click the "View" tab. This is the section we need to be in to tell it to not hide the extension types. You should see the proper option listed as "Hide extensions for known file types", uncheck that option and then click the OK button. You should now see "main.cs".




Now that we can see our .cs files, we can also create them easily. To do this simply right click on an empty spot on the screen and click File->New. You should now see a menu drop down. Select Text Document from the list. Now you should see a new text document called "New Text Document.txt". The text is highlighted so you can type a new name. Type in the name "test.cs" and press enter. You should be prompted with a yes or no box asking if you want to change your extension. Click yes and you should now see your new file with the proper .cs extension to be a T2D script file.







Now that we have a script file, we can edit it by right clicking on the file and selecting Open With. If you see "Notepad" listed then select that. If you don't see "Notepad" listed then click "Choose Program". This will present you with another list. From here you can choose "Notepad". It should then open up your blank script file in Notepad. This is how you can create and edit script files for the upcoming tutorials. There are many other script editing programs, such as Torsion.



Mac Scripting


To create script files on a Mac you can go to Applications/TextEdit. You should now be presented with a basic text editor. Script files are in plain text and by default you cannot save to plain text, so to change this you can click Format->Make Plain Text. Now we can click File->Save As. You should be presented with a Save dialog. Be sure to save the name with a ".cs" extension (such as "test.cs"). Then cilck the blue arrow to bring up the file browser. The first thing you need to do is uncheck the "hide extension" check box. Now we need to save this empty script file in our game directory. Browse to wherever you saved your project. When you have chosen the folder to save it to, click the Save button and you should be presented with a dialog asking you to change your file extension to .txt or append it with .txt. Don't append it and make sure you use .cs.
