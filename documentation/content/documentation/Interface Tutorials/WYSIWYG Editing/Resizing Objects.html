1.1 Resizing Objects

Resizing Objects

To resize an object in the LevelBuilder, first select the object. (as shown in Figure 1.1.1).

Figure 1.1.1

Next, click and drag on one of the blue squares to resize it (as shown in Figure 1.1.2).
Figure 1.1.2

You can control how the image is transformed by holding different keys.
  • Control - Resizes the object relative to the center (leaving the center in the same place)
  • Shift - keeps the object's aspect ratio the same
  • Alt - rotates the object
