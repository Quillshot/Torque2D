Importing Resources

To import a T2D resource pack into your project, click the “Project->Resources...” dropdown menu. You will be presented with two lists. The list on the left contains all of the available resources that we don't have linked into your project, while the list on the right contains all of the resources currently linked into your project. For example, if you want to use the art from the checkers demo, select the “checkersArt” resource in the left list and click the Add To Project button (as shown in Figure 1). Now you should see the checkersArt resource listed in the right list. Click the Save button to finish the process (as shown in Figure 2). Now your object library in the right pane should be populated with checkers art (as shown in Figure 3). With a couple clicks we've imported all the art needed for checkers!
