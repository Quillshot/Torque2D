New Assets

A few of the tutorials in the documentation instruct you to use images and particle effects you may not have. Most likely, you are being asked to use assets that have been deprecated in this version of T2D, but the tutorials have not been updated yet.

This mostly occurs in the TutorialBase project, so use the following instructions to get those effects and images in your local TutorialBase.

Click here to download a zip file containing the additional effects. Inside the zip file, you will find a folder called images and a script file called resourceDatabase.cs. The folder contains all the base images used to create the additional imageMaps and particle effecst you are missing.

The script contains the datablocks used to build the images within your game. The script is actually the same resourceDatabase.cs found in the stock TutorialBase base project, but I've added additional datablocks to make sure of the new imageMaps and particle effects.


Automatic Merging

If you have not made any changes to your TutorialBase project, or any other project that uses the starterArt resource, then you can simply drag and drop the images folder and resourceDatabase.cs script.


Manual Merge

With these files, you can either perform a manual merge or simply copy the files over your existing ones. If you are trying to add these resources to a project other than TutorialBase, or you've modified TutorialBase to contain new imageMaps, you should perform a manual merge. Here are the following steps you should follow to perform a manual merge:

  1. Open up the zip file.
  2. Copy the images from the zip file to your TutorialBase\resources\starterArt\images folder. Make sure you do not copy over any images you already have that share the same name.
  3. Open resourceDatbase.cs found in TutorialBase\resources\starterArt with your text editor of choice
  4. Copy the code found below these instructions, then paste it at the bottom of your script file just before the final }; bracket.
  5. Save your script file run your project to make sure the images show up.
