Camera Sizing

The camera is represented by a dark blue box that you should see in your level (as shown in Figure 1).
Figure 1.
There are two ways to configure the camera size, we can click the camera tool icon (in the top menu bar) and visually resize it (just like the world limit editing tool), or we can specify the width and height manually as values. To set the camera to a specific width and height, you click the Project tab in the right panel and then click the “t2dSceneGraph” object in the Object Tree view (as shown in Figure 2).
Figure 2
Now you have the “t2dSceneGraph” selected, click the “Edit” tab and then change the camera width and height to whatever you want and then press enter (as shown in Figure 3).


Figure 3
You should see the dark blue box expand, possibly beyond your current view. You may need to zoom out to view our new camera size (as shown in Figure 4). You can zoom out by either using the scroller if you have a scroll mouse or, if not, you can go to the “View” dropdown menu and choose an appropriate zoom percentage.
Figure 4
If you remember, the previous width and height was 100 and 75. In Torque 2D the actual size numbers are relative. What this means is you can set it to 4 and 3 if you wanted, or 4000 and 3000. We changed ours to 800 by 600 so we can get a better idea of our image and screen resolution in comparison to our images and values. T2D maintains its world sizing despite the screen resolution of the computer playing a T2D game, which, as you can imagine, is a very good thing.
