Making a New Scene

To make a new scene, click the page icon in the top left of the Scene Editor (as shown in Figure 1.1.1).


Figure 1.1.1

Or, you can go to the File menu and select New Scene (as shown in Figure 1.1.2).

Figure 1.1.2

You will have to save a new scene before playing it, unless you are testing it from within the Scene Builder. If you click the Play Scene button or the Project->Run Game menu item the current scene will run in a test window without having to explicitly save the scene.
