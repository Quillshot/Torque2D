Saving Your Scene

To save your scene, click the disk icon in the top left of the Scene Editor (as shown in Figure 1.1.1).


Figure 1.1.1 

Or, you can go to the File menu and select Save (as shown in Figure 1.1.2).


Figure 1.1.2

Saving a scene puts it in the "data/levels" folder of your project.
