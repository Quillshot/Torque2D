Loading Levels

To load a scene within the current project, click the folder icon in the top left of the Scene Editor (as shown in Figure 1.1.1).


Figure 1.1.1

Or, you can go to the File menu and select Open Scene. (as shown in Figure 1.1.2).


Figure 1.1.2

After doing one of the above, you will be shown a dialog box where you can select the scene file from your project's folder structure to open.
