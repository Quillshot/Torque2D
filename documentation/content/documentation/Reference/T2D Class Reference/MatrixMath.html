MatrixMath Description: Matrix manipulation functions.

Methods:

MatrixCreate(Vector3F pos, Vector3F rot)

Description: Create a matrix representing the given translation and rotation.

MatrixCreateFromEuler(Vector3F e)

Description: Create a matrix from the given rotations.

MatrixMulPoint(MatrixF xfrm, Point3F pnt)

Description: Multiply pnt by xfrm.

MatrixMultiply(Matrix4F left, Matrix4F right)

Description: Multiply the two matrices.

MatrixMulVector(MatrixF xfrm, Point3F vector)

Description: Multiply the vector by the transform.
