t2dVectorMath Description: T2D Vector manipulation functions.

Methods:

t2dAngleBetween(t2dVector v1 $, t2dVector v2 $)

Description: Returns the angle between v1 and v2.

t2dRectNormalise(t2dVector v1 $, t2dVector v2 $)

Description: Returns normalise rectangle of v1 and v2.

t2dVectorAdd(t2dVector v1 $, t2dVector v2 $)

Description: Returns v1+v2.

t2dVectorCompare(t2dVector p1 $, t2dVector p2 $,[epsilon])

Description: Compares points p1 and p2 with optional difference (epsilon).

t2dVectorDistance(t2dVector p1 $, t2dVector p2 $)

Description: Returns the distance between points p1 and p2.

t2dVectorDot(t2dVector v1 $, t2dVector v2 $)

Description: Returns dot-product of v1 and v2.

t2dVectorLength(t2dVector v1 $)

Description: Returns the length of v1.

t2dVectorMult(t2dVector v1 $, t2dVector v2 $)

Description: Returns v1 mult v2.

t2dVectorNormalise(t2dVector v1 $)

Description: Returns normalised v1.

t2dVectorScale(t2dVector v1 $, scale)

Description: Returns v1 scaled by scale.

t2dVectorSub(t2dVector v1 $, t2dVector v2 $)

Description: Returns v1-v2.
