Redbook Description: Control functions for Redbook audio (ie, CD audio).

Methods:

redbookClose()

Description: Close the current Redbook device.

redbookGetDeviceCount()

Description: get the number of redbook devices.

redbookGetDeviceName(int index)

Description: Get name of specified Redbook device.

redbookGetLastError()

Description: Get a string explaining the last redbook error.

redbookGetTrackCount()

Description: Return the number of tracks.

redbookGetVolume()

Description: Get the volume.

redbookOpen(string device=NULL)



redbookPlay(int track)

Description: Play the selected track.

redbookSetVolume(float volume)

Description: Set playback volume.

redbookStop()

Description: Stop playing.
