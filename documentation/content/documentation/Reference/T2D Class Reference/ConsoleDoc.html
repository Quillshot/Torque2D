ConsoleDoc Description: Console self-documentation functions. These output psuedo C++ suitable for feeeding through Doxygen or another auto documentation tool.

Methods:

dumpConsoleClasses(bool dumpScript=true, bool dumpEngine=true)

Description: dumps all declared console classes to the console. Params: dumpScript: Specifies whether or not classes defined in script should be dumped. dumpEngine: Specifies whether or not classes defined in the engine should be dumped.

dumpConsoleFunctions(bool dumpScript=true, bool dumpEngine=true)

Description: Dumps all declared console functions to the console. Params: dumpScript: Specifies whether or not functions defined in script should be dumped. dumpEngine: Specifies whether or not functions defined in the engine should be dumped.
