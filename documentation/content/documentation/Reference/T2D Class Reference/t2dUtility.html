t2dUtility Description: T2D Utility functions.

Methods:

addBitToMask(mask, bit)

Description: Returns the mask with a bit added to it

bit()

Description: Converts a bit-position into a value.

bitInverse()

Description: Returns the ones complement of a bit.

bits()

Description: Converts a list of bit-positions into a value.

getT2DMaxTextureSize()

Description: Returns the maximum texture size supported by the current hardware.

getT2DVersion()

Description: Returns T2D Version

removeBitFromMask(mask, bit)

Description: Returns the mask with a bit removed from it

t2dAssert(condition, message)

Description: Fatal Script Assertion

t2dAssertISV(condition, message)

Description: Fatal ISV Script Assertion

t2dBeginScene()

Description: Sets the default Scene Graph.

t2dEndScene()

Description: Resets the default Scene Graph.

t2dGetMax(a, b)

Description: Returns the Maximum of two values.

t2dGetMin(a, b)

Description: Returns the Minimum of two values.
