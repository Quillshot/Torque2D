VectorMath Description: Vector manipulation functions.

Methods:

VectorAdd(Vector3F a, Vector3F b)

Description: Returns a+b.

VectorCross(Vector3F a, Vector3F b)

Description: Calculate the cross product of a and b.

VectorDist(Vector3F a, Vector3F b)

Description: Calculate the distance between a and b.

VectorDot(Vector3F a, Vector3F b)

Description: Calculate the dot product of a and b.

VectorLen(Vector3F v)

Description: Calculate the length of a vector.

VectorNormalize(Vector3F a)

Description: Returns a scaled such that length(a) = 1.

VectorOrthoBasis(AngAxisF aaf)

Description: Create an orthogonal basis from the given vector. Return a matrix.

VectorScale(Vector3F a, float scalar)

Description: Returns a scaled by scalar (ie, a*scalar).

VectorSub(Vector3F a, Vector3F b)

Description: Returns a-b.
