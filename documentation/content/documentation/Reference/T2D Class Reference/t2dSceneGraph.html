onUpdateScene(%this)

Brief Description: A frame has completed processing. Description: This is called every frame, and thus is dependent on the frame rate of the engine. Therefore, you should not do any movement of objects here, because the speed at which they move will not be constant. Also, it is recommended to only use this callback on a specific scenegraph. Using it on the entire t2dSceneGraph class can result in LevelBuilder/Packaged Game differences. Params: this:
