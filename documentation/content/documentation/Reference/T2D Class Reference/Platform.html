Platform Description: General platform functions.

Methods:

getRealTime()

Description: Return the current real time in milliseconds.

getSimTime()

Description: Return the current sim time in milliseconds.

lockMouse(bool isLocked)

Description: Lock the mouse (or not, depending on the argument's value) to the window.

playJournal(string filename, bool break=false)

Description: Begin playback of a journal from a specified field, optionally breaking at the start.

saveJournal(string filename)

Description: Save the journal to the specified file.

setNetPort(int port)

Description: Set the network port for the game to use.
