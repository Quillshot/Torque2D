Torque 2D (T2D) Documentation Overview

As a T2D user, you have access to a significant amount of documentation and tutorials. This page offers a quick overview of what's available.

All told, T2D ships with over 300 pages of documentation right in the SDK! And since it's all in html format, you can now cross-reference all the functions and glossary terms with just a click!

Getting Started


With the amount of documentation here it can be hard finding a place to start. We recommend reading the Introduction to get an overview of the product before moving to the Tutorials.  The Tutorials are ordered in the recommended completion order. The best place to start is with the Fish Demo Tutorial and the Fish Game Tutorial. Also be sure to check out the Feature Tutorials for information on specific functionality within the engine. When you've achieved a basic understanding of scripting in T2D then trying out the Behavior Tutorials is a great next step to learning the newest and one of the most useful functionality in T2D.
